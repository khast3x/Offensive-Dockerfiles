# Tulpar
> Web Vulnerability Scanner


## Source

Based on [this project](https://github.com/anilbaranyelken/tulpar)  

## Usage:  

```bash
cd tulpar/
docker build -t tulpar .
docker run -it tulpar:latest
```
## Help
```bash
#Use tool:
docker run -it tulpar action http://target.com
```
